Wherein are listed the commands at your disposal.

You can get to this page at any time using the "list ubiquity commands" command.

You have commands in command feeds. | You have unsubscribed from command feeds. |

  • All the commands that Ubiquity recognizes are listed in the right column. The black text is the command's name (what you type into Ubiquity to use it). The grey text explains what the command does.
  • The left column shows all the "feeds" that Ubiquity is subscribed to. Each feed can contain multiple commands.
  • You can view this page sorted by feeds or sorted by commands. Click any of the links next to "sort by", below.
  • If there's a command you don't want, uncheck the box next to it. You can get the command back by re-checking the box.
  • If there's a whole command feed you don't want, click the "[unsubscribe]" link beneath the feed name. All of the commands in the feed will stop working. If you change your mind, you can re-subscribe at the bottom of the page.
